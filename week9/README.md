# [Week 9] 뉴스 감정 분석 & Streamlit / 허깅페이스

**주제**
- 뉴스 텍스트 데이터 기반 감정 분석 및 웹 애플리케이션 구현

**사용 기술/라이브러리**
- Python, Pandas, Hugging Face Transformers, Streamlit, Matplotlib, Seaborn

**목표/문제 정의**
- 뉴스 기사 텍스트에서 긍정/부정/중립 감정을 분석
- Streamlit을 이용해 분석 결과를 시각화하고 웹 앱으로 제공

**데이터 전처리**
- 텍스트 정제 및 토큰화
- 불용어 제거 및 필요 없는 기호 제거
- 학습/테스트 데이터 분리

**모델링 / 분석 과정**
- Hugging Face 사전 학습 모델(Fine-tuning 없이) 적용
- 뉴스 기사별 감정 예측 수행
- Streamlit을 통해 결과 대시보드 생성

**결과/성과**
- 기사별 감정 분류 자동화
- 시각화를 통한 뉴스 감정 트렌드 확인 가능
- 간단한 웹 앱으로 데이터 분석 결과 공유 가능

**학습 포인트**
- NLP 사전 학습 모델 활용 실습
- Streamlit을 이용한 데이터 시각화 및 웹 앱 구현 경험
- 텍스트 데이터 기반 감정 분석 이해
