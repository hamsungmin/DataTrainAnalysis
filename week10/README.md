# [Week 10] PDF 활용 FAQ 검색 모델 생성

**주제**
- PDF 자료를 활용한 FAQ 검색 모델 구축

**사용 기술/라이브러리**
- Python, Pandas, PyPDF2 / pdfplumber, scikit-learn, NLP 라이브러리, TF-IDF, cosine similarity

**목표/문제 정의**
- PDF 문서에서 질의와 관련된 답변을 자동으로 검색
- 유사 문서 검색 및 FAQ 모델 구현

**데이터 전처리**
- PDF 텍스트 추출 및 정제
- 문장 단위 분리 및 토큰화
- 불용어 제거 및 벡터화(TF-IDF)

**모델링 / 분석 과정**
- TF-IDF 벡터 생성
- Cosine Similarity 기반 질의-문서 매칭
- 질문 입력 시 가장 유사한 답변 반환

**결과/성과**
- PDF 내 정보 검색 자동화 가능
- FAQ 검색 결과 정확도 향상
- 유사 질문에 대한 신속한 답변 제공

**학습 포인트**
- 문서 데이터 처리 및 벡터화 방법 학습
- 텍스트 기반 검색 모델 구현 경험
- NLP 기초와 유사도 계산 이해
