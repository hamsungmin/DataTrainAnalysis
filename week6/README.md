# [Week 6] 보스턴 집값 예측

**주제**
- 보스턴 주택 가격 데이터셋을 활용한 회귀 모델 학습 및 예측

**사용 기술/라이브러리**

Python, Pandas, NumPy, scikit-learn, Matplotlib, Seaborn

**목표/문제 정의**
- 주어진 보스턴 주택 데이터에서 집값(MEDV)을 예측
- 다양한 회귀 모델을 비교하고 최적 모델 선정

**데이터 전처리**
- 결측치 처리 및 데이터 정규화
- 변수 간 상관관계 분석
- 카테고리형 변수 인코딩

**모델링**
- 선형 회귀(Linear Regression)
- 릿지(Ridge) / 라쏘(Lasso) 회귀
- 결정 트리(Decision Tree)
- 랜덤 포레스트(Random Forest)

**모델 평가**
- R², RMSE 등의 지표 사용
- Cross-validation 적용

**결과/성과**
- 랜덤 포레스트 모델에서 가장 높은 예측 성능 달성
- 변수 중요도 시각화를 통해 가격에 영향을 많이 미치는 요인 확인

**학습 포인트**
- 회귀 모델 비교 방법 학습
- 데이터 전처리 및 시각화 중요성 경험
- scikit-learn 모델 활용법 강화
