# [Week 8] GPT 활용 고객 피드백 분석

**주제**
- GPT 모델을 활용한 고객 피드백 텍스트 분석

**사용 기술/라이브러리**
- Python, Pandas, OpenAI GPT API, Matplotlib, Seaborn, NLP 라이브러리

**목표/문제 정의**
- 고객 리뷰 및 피드백 데이터를 분석하여 주요 의견과 감정 파악
- GPT 모델을 이용한 텍스트 요약 및 감정 분석

**데이터 전처리**
- 결측치 제거 및 텍스트 정제
- 토큰화(Tokenization) 및 불용어 제거
- 데이터 샘플링 및 라벨링(감정 긍정/부정)

**모델링 / 분석 과정**
- GPT API를 이용한 리뷰 요약
- 감정 분석(Sentiment Analysis) 수행
- 주제별 클러스터링 및 시각화

**결과/성과**
- 고객 피드백 핵심 내용 자동 요약 가능
- 긍정/부정 감정 비율 및 트렌드 시각화
- 특정 제품/서비스 개선 포인트 도출

**학습 포인트**
- GPT 모델 활용 실습 및 API 연동 경험
- 텍스트 데이터 전처리 및 NLP 분석 방법 학습
- 시각화와 데이터 인사이트 도출 능력 강화
