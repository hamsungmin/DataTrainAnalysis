# [Week 3] 데이터 분석 및 시각화 프로젝트

**프로젝트 제목**
- Pandas와 Matplotlib/Seaborn을 활용한 데이터 분석 & 시각화 실습

**핵심 주제**
- 실전 데이터 분석(EDA)
- 통계 기반 탐색
- 다양한 시각화 기법 적용
- 데이터 인사이트 도출

**사용 데이터**
- CSV 파일(판매 데이터, 사용자 행동 데이터, 매출 데이터 등 교육 제공 예시)
- 2~3개 이상의 컬럼을 가진 실제 데이터셋 형태

**사용 기술 / 라이브러리**
- pandas
- numpy
- matplotlib
- seaborn
- datetime 처리

**분석·시각화 내용 요약**
- 데이터 불러오기 및 기본 구조 확인
- 결측치/이상치 처리
- 기초 통계 분석(평균, 분산, 상관관계 등)
- 범주형/연속형 데이터별 시각화
    - 바차트
    - 히스토그램
    - 박스플롯
    - 라인차트
    - 상관계수 히트맵
- 특정 패턴/트렌드 찾기
- 간단한 인사이트 도출(예: 특정 요일 매출 증가, 특정 제품군 이상치 등)

**결과/성과**
- 데이터 전처리 → 시각화 → 인사이트 도출까지 “데이터 분석 Full Pipeline” 완성
- 이후 ML 모델링(Week 4~7)의 기반이 되는 분석 역량을 체득
