# [Week 1] 파이썬 기본 프로젝트

**프로젝트 제목**
- 파이썬 기초 문법 실습 및 데이터 처리 기본 연습

**핵심 주제**
- Python 기본 문법
- 리스트/딕셔너리 활용
- 조건문/반복문 기초
- 간단한 데이터 처리 흐름 구현

**사용 데이터**
- 간단한 예시 데이터(수치형, 문자열 리스트 등)
- 외부 데이터 없음 (기초 문법 중심)

**사용 기술 / 라이브러리**
- Python
- 기본 함수/반복문/조건문
- 파일 입출력 일부 포함될 가능성 있음

**분석/코드 내용 요약**
- 변수, 함수, 리스트, 딕셔너리 등을 활용한 기초 로직 구현
- 문자열 처리, 리스트 순회 등 알고리즘 기초
- 기초적인 데이터 가공 예제 포함
- 교육용 Step-by-step Python 기본 예제 형태

**결과/성과**
- Python을 활용한 기본 데이터 처리 흐름을 익히는 입문 단계
- 이후 Week 2~16까지 이어지는 “데이터 분석 사전 워밍업 프로젝트” 역할
