# [Week 2] 업무 자동화 프로젝트

**프로젝트 제목**
- Python을 활용한 업무 자동화 실습

**핵심 주제**
- 반복적인 사무 업무를 Python으로 자동화
- 파일 처리, 폴더 정리, 엑셀/CSV 자동 처리
- 간단한 데이터 수집 또는 데이터 변환 자동화 흐름 구현

**사용 데이터**
- CSV/Excel, 텍스트 파일
- 업무 자동화 예제로 제공된 샘플 데이터

**사용 기술 / 라이브러리**
- pandas (엑셀/CSV 자동화)
- os / glob (파일 탐색·정리)
- openpyxl 또는 csv
- 경우에 따라 requests (웹 요청 자동화)

**분석/업무 자동화 내용 요약**
- 특정 폴더 내 파일을 읽어 하나의 파일로 자동 통합
- 데이터 전처리(결측치 처리, 컬럼 정리) 자동 처리
- 파일명 규칙에 따라 자동 재정렬/자동 저장
- 일정한 형식의 보고서나 출력물 자동 생성
- 반복되는 수작업을 코드로 대체하는 업무 자동화 패턴 실습

**결과/성과**
- 직접 실행 가능한 "업무 자동화 스크립트" 완성
- 사람이 반복적으로 해야 하는 작업(PPT/CSV 병합, 파일 정리 등)을 자동 처리
- 이후 데이터 분석 프로젝트에서 데이터 준비 시간을 크게 단축할 수 있는 기반 확보
