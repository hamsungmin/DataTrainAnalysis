# [Week 13] PDF 자료 분석 & OCR 처리

**주제**
- PDF 자료에서 OCR을 활용한 텍스트 추출 및 분석

**사용 기술/라이브러리**
- Python, PyPDF2 / pdfplumber, Tesseract OCR, OpenCV, Pandas

**목표/문제 정의**
- 스캔된 PDF 또는 이미지 PDF에서 텍스트 데이터를 추출
- 추출된 데이터를 분석하여 유용한 정보 도출

**데이터 전처리**
- PDF 파일에서 이미지 및 텍스트 추출
- OCR 처리로 이미지 내 텍스트 변환
- 텍스트 정제 및 토큰화

**모델링 / 분석 과정**
- OCR로 텍스트 데이터화
- Pandas로 데이터 프레임 구성
- 키워드 추출, 통계 분석 및 시각화

**결과/성과**
- PDF 자료에서 텍스트 자동 추출 가능
- OCR 정확도 향상 및 데이터 정제 수행
- 분석 및 시각화를 통한 정보 활용

**학습 포인트**
- OCR 처리 및 PDF 데이터 활용 경험
- 이미지 기반 텍스트 추출과 데이터 정제 이해
- 실무 데이터 전처리 과정 실습
