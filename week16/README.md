# [Week 16] 랭스미스를 이용한 랭체인 분석

**주제**
- LangSmith를 활용한 LangChain 분석 및 모델 디버깅 실습

**사용 기술/라이브러리**
- Python, LangChain, LangSmith, OpenAI GPT API, Pandas, NLP 라이브러리

**목표/문제 정의**
- LangChain 체인을 모니터링하고 분석하여 성능 개선
- LangSmith를 통한 체인 동작 시각화 및 디버깅 경험

**데이터 전처리**
- LangChain 입력/출력 데이터 수집
- 로그 및 결과 데이터 정제
- 분석용 데이터프레임 구성

**모델링 / 분석 과정**
- LangChain 체인 실행 및 추적
- LangSmith 대시보드를 통해 체인 분석
- 문제 발생 시 디버깅 및 개선

**결과/성과**
- 체인 실행 흐름 시각화 및 분석 완료
- 문제 발생 원인 파악 및 모델 개선 가능
- LangChain 체인 운영 이해도 향상

**학습 포인트**
- LangChain 구조와 LangSmith 활용법 이해
- 체인 모니터링 및 디버깅 경험
- 실무에서 체인 성능 분석 및 개선 능력 강화
