# [Week 15] RAG 체인 구성

**주제**
- RAG(Retrieval-Augmented Generation) 체인을 활용한 문서 기반 질의응답 모델 구축

**사용 기술/라이브러리**
- Python, LangChain, OpenAI GPT API, FAISS, Pandas, NLP 라이브러리

**목표/문제 정의**
- 문서 데이터에서 필요한 정보를 검색하고 GPT 모델을 통해 답변 생성
- 검색 기반 생성(RAG) 모델 구조 이해 및 구현

**데이터 전처리**
- 문서 텍스트 추출 및 정제
- 문서 단위 분할 및 인덱싱
- 벡터화 및 검색용 데이터 구조 준비

**모델링 / 분석 과정**
- RAG 체인 구성: 검색 + 생성 통합
- 질문 입력 시 관련 문서 검색 후 GPT를 통해 답변 생성
- 검색 결과와 생성 결과 비교 및 검증

**결과/성과**
- 문서 기반 질의응답 자동화 가능
- RAG 구조 이해 및 실습 완료
- 검색과 생성 통합 처리 경험

**학습 포인트**
- RAG 체인 구조 이해
- 문서 검색과 생성 모델 연동 경험
- 실무에서 문서 기반 AI 응용 가능성 체험
