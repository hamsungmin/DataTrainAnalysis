# [Week 4] 구매 트렌드 분석 프로젝트

**프로젝트 제목**
- 고객 구매 데이터 기반 트렌드 분석

**핵심 주제**
- 고객 구매 패턴 분석
- 상품별/기간별 판매 변화 탐색
- 데이터 기반 소비 트렌드 파악

**사용 데이터**
- CSV 형태의 구매 기록 데이터
- 날짜, 카테고리, 상품명, 구매수량, 금액 등 포함된 구조

**사용 기술 / 라이브러리**
- pandas
- numpy
- matplotlib
- seaborn
- datetime

**분석 내용 요약**
- 날짜 기반 데이터 전처리(일/월 단위 집계)
- 제품 카테고리별 판매량 비교
- 월별/주별 트렌드 라인 분석
- 매출 상위 제품 Top N 분석
- 특정 날짜 또는 시즌 변화 탐색
- 고객 세그먼트 분석이 포함될 수도 있음

**결과/성과**
- 구매 패턴의 명확한 시즌성 또는 증감 추세 확인
- 어떤 제품군이 매출을 견인하는지 파악
- “언제, 무엇이, 얼마나” 팔리는지 핵심적 인사이트 도출
- 비즈니스 의사결정을 지원하는 데이터형 분석 결과 생성
