# [Week 7] 주식 데이터 딥러닝

**주제**
- 주식 가격 데이터 기반 딥러닝 모델을 활용한 예측

**사용 기술/라이브러리**
- Python, Pandas, NumPy, TensorFlow / Keras, Matplotlib

**목표/문제 정의**
- 과거 주식 데이터를 활용하여 다음 날 주식 가격 또는 추세를 예측
- 딥러닝 모델 적용을 통한 성능 비교

**데이터 전처리**
- 결측치 처리 및 결측 데이터 보간
- 정규화(Normalization) 적용
- 시계열 데이터 형태로 변환(LSTM 입력용)

**모델링**
- LSTM(Long Short-Term Memory) 모델
- Dense 레이어 추가하여 출력 예측
- 하이퍼파라미터 튜닝

**모델 평가**
- MSE(Mean Squared Error) 및 RMSE 사용
- 학습/검증 데이터 분리 및 모델 과적합 확인

**결과/성과**
- LSTM 모델로 주식 가격의 단기 추세 예측 가능
- 시각화를 통해 실제값과 예측값 비교

**학습 포인트**
- 시계열 데이터 처리 방법 학습
- 딥러닝 모델(LSTM) 구조와 활용법 이해
- 하이퍼파라미터 조정과 모델 성능 분석 경험
