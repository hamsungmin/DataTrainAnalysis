# [Week 12] YOLO 객체 인식

**주제**
- YOLO(You Only Look Once) 모델을 활용한 객체 인식 실습

**사용 기술/라이브러리**
- Python, OpenCV, PyTorch, YOLO, Matplotlib

**목표/문제 정의**
- 이미지 및 동영상에서 실시간 객체 인식 수행
- 다양한 클래스에 대한 객체 탐지 및 위치 표시

**데이터 전처리**
- 이미지 크기 조정 및 정규화
- 클래스 라벨링 및 학습용 데이터셋 준비
- 데이터 증강(Augmentation) 적용

**모델링 / 분석 과정**
- YOLO 사전 학습 모델 사용
- Fine-tuning 없이 테스트 이미지/영상에 객체 탐지
- 탐지 결과 시각화 및 정확도 확인

**결과/성과**
- 실시간 객체 탐지 가능
- 다양한 클래스 객체 인식 성공
- 탐지 결과 이미지/영상으로 시각화

**학습 포인트**
- YOLO 모델 구조 및 활용 방법 이해
- 객체 인식과 위치 좌표 처리 경험
- OpenCV와 PyTorch 연동 실습
