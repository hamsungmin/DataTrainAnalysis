# [Week 11] 이미지 추론 샘플

**주제**
- 이미지 데이터를 활용한 딥러닝 추론 실습

**사용 기술/라이브러리**
- Python, TensorFlow / Keras, OpenCV, Matplotlib, PIL

**목표/문제 정의**
- 이미지 데이터셋을 기반으로 객체/패턴 인식 모델 학습 및 추론 수행
- 딥러닝 모델을 활용한 이미지 예측 실습

**데이터 전처리**
- 이미지 크기 조정 및 정규화
- 데이터 증강(Augmentation) 적용
- 학습/검증 데이터 분리

**모델링 / 분석 과정**
- CNN(Convolutional Neural Network) 모델 설계

**학습 및 검증 수행**
- 테스트 이미지에 대한 추론 결과 확인

**결과/성과**
- 이미지 분류 및 패턴 인식 성공
- 학습된 모델로 새로운 이미지 추론 가능
- 시각화를 통해 정확도 및 학습 과정 확인

**학습 포인트**
- CNN 구조와 이미지 데이터 처리 경험
- 이미지 전처리 및 증강(Augmentation) 이해
- 딥러닝 추론 과정 실습
