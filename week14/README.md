# [Week 14] 데이터 활용한 AI 챗봇 생성

**주제**
- 데이터 기반 AI 챗봇 구축 및 실습

**사용 기술/라이브러리**
- Python, Pandas, OpenAI GPT API, Flask / Streamlit, NLP 라이브러리

**목표/문제 정의**
- 데이터셋을 기반으로 자연어 질문에 답변할 수 있는 챗봇 생성
- 간단한 웹 인터페이스를 통해 챗봇 테스트

**데이터 전처리**
- 텍스트 데이터 정제 및 토큰화
- 질문-답변 쌍 구성
- 불용어 제거 및 벡터화

**모델링 / 분석 과정**
- GPT API를 활용한 챗봇 답변 생성
- 웹 인터페이스(Flask/Streamlit) 연결
- 질의에 대한 실시간 응답 확인

**결과/성과**
- 데이터 기반 AI 챗봇 구현 완료
- 질문 입력 시 관련 답변 자동 제공
- 웹 인터페이스를 통한 테스트 및 시연 가능

**학습 포인트**
- 챗봇 구조와 GPT API 연동 경험
- 데이터 전처리 및 질문-답변 구성 방법 학습
- 웹 인터페이스 연동 및 실시간 테스트 경험
